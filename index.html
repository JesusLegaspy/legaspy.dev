Museum of Candy

HTML5 Bootstrap CSS JavaScript

The museum of candy site is an exercise of making a responsive web page. On wider screens, the layout has an alternating text-picture patter. While on narrower screens, the text-picture pattern is preserved by a clever Bootstrap trick on changing the order. The top menu bar changes size depending on screen width. A little bit of JavaScript magic is used to transition the top menu bar color when the user scrolls the page. The page includes fonts from Google to complement the aesthetic.
